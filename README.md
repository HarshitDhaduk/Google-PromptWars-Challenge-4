# Stadium Copilot — Smart Stadiums & Tournament Operations

A context-aware, multilingual GenAI assistant for fans at the FIFA World Cup 2026 Final
(MetLife Stadium, July 19, 2026). Built for PromptWars Challenge 4.

> Work in progress — full documentation lands with the final commit.

## Stack

- `web/` — Next.js 15 (App Router, TypeScript, Tailwind CSS)
- `service/` — Python FastAPI service owning all GenAI integration (Google Gemini)
